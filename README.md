# Bape
A Bathing Ape  

# Browser Support
Sorry ~ Only support the modern browsers

# Installation
`npm i bape --save`

# API
* `cookie`
* `formatJSON(value, [replacer], [space])`
* `isEmptyObject(input)`
* `isEnter(e)`
* `isJSON(input)`
* `isObject(input)`
* `isPlainObject(input)`
* `isURL(input)`

# Dependency
* [js-cookie](https://github.com/js-cookie/js-cookie)
